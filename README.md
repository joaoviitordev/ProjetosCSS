# Projetos HTML & CSS 🎨

Repositório com projetos simples desenvolvidos em **HTML e CSS**, com foco no **estudo e prática de CSS moderno**, layout e responsividade.

## Objetivo
Aprimorar habilidades em:
- HTML5
- Estilização avançada com CSS
- Boas práticas de front-end

## Projetos
Os projetos abordam conceitos como:
- Flexbox e Grid
- Responsividade
- Animações e transições
- Organização e reutilização de estilos

## Tecnologias
- HTML5
- CSS3

## Observação
Projetos com foco educacional, voltados para aprendizado contínuo e construção de portfólio front-end.
